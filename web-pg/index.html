What is L-Edger?

L-Edger is a powerful Flutter-based link management application that helps you organize and access your important links effortlessly. Create folders, search by keywords, and copy links to your clipboard with just a double tap.

Installation Guide

For Users:

  1. Download the APK using the button above
  2. Open the APK on your Android device
  3. Allow installation from unknown sources if prompted
  4. Follow the on-screen instructions to complete installation

For Developers:

  1. Clone the repository: git clone https://github.com/dwukn/l-edger.git
  2. Navigate to the project directory: cd l-edger
  3. Install dependencies: flutter pub get
  4. Run the app: flutter run
